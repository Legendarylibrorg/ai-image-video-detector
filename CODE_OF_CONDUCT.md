# Code Of Conduct

## Our Standard

Please be respectful, constructive, and kind.

Examples of expected behavior:

- Give actionable technical feedback without personal attacks.
- Assume good intent and ask clarifying questions before escalating.
- Keep discussions focused on the code, docs, tests, and user impact.
- Help new contributors understand repo conventions.

Examples of unacceptable behavior:

- Harassment, insults, or intimidation.
- Publishing private information without permission.
- Abusive language, discriminatory remarks, or repeated bad-faith disruption.

## Scope

This applies to repository discussions, issues, pull requests, review comments, and other project spaces.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers through GitHub moderation/reporting tools or use the private security/contact path described in [SECURITY.md](SECURITY.md) when appropriate.

## Enforcement

Maintainers may remove comments, close discussions, or block participation when behavior violates this code of conduct.
